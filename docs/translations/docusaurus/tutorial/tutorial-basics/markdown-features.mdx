# 마크다운 기능

Docusaurus는 **[마크다운](https://daringfireball.net/projects/markdown/syntax)**과 몇 가지 **추가 기능**을 지원합니다.

## 프론트 매터

마크다운 문서의 상단에는 [프론트 매터](https://jekyllrb.com/docs/front-matter/)라는 메타데이터가 있습니다:

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: 내 문서 제목
description: 내 문서 설명
slug: /my-custom-url
---

## 마크다운 헤딩

[링크](./hello.md)가 포함된 마크다운 텍스트
```

## 링크

일반적인 마크다운 링크는 URL 경로나 상대 파일 경로를 사용하여 지원됩니다.

```md
페이지를 [생성하는 방법](/create-a-page)을 살펴보세요.
```

```md
페이지를 [만드는 방법](./create-a-page.md)을 살펴보세요.
```

**결과:** [페이지 생성 방법](./create-a-page.md)을 살펴보세요.

## 이미지

일반적인 마크다운 이미지를 지원합니다.

정적 디렉토리(`static/img/docusaurus.png`)의 이미지를 참조하기 위해 절대 경로를 사용할 수 있습니다:

```md
![도큐사우루스 로고](/img/docusaurus.png)
```

![Docusaurus logo](/img/docusaurus.png)

또한 현재 파일에 대한 상대 경로를 사용하여 이미지를 참조할 수 있습니다. 이는 마크다운 파일을 사용하여 이미지를 가까이 배치하는 데 특히 유용합니다:

```md
![Docusaurus logo](./img/docusaurus.png)
```

## 코드 블록

마크다운 코드 블록은 구문 강조 표시를 지원합니다.

````md
```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```
````

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
  return <h1>Hello, Docusaurus!</h1>;
}
```

## 권고사항

Docusaurus는 권고사항과 콜아웃을 만들기 위한 특수 구문을 지원합니다:

```md
:::tip 나의 팁

이 좋은 기능 옵션을 사용하세요

:::

:::danger 주의하세요

이 작업은 위험합니다

:::
```

:::tip 나의 팁

이 좋은 기능 옵션을 사용하세요

:::

:::danger 주의하세요

이 작업은 위험합니다

:::

## MDX와 React 컴포넌트

[MDX](https://mdxjs.com/)는 문서를 더 상호작용적으로 만들고, 마크다운 내부에 어떤 **React 컴포넌트**도 사용할 수 있게 합니다:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`당신은 ${children} 레이블이 있는 색상 ${color}을(를) 클릭했습니다`)
    }}>
    {children}
  </span>
);

이것은 <Highlight color="#25c2a0">Docusaurus 초록색</Highlight> 입니다!

이것은 <Highlight color="#1877F2">Facebook 파란색</Highlight> 입니다!
```
