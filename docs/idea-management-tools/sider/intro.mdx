# Sider 소개

Sider는 **인공지능 웹브라우저 도우미**로, 웹페이지에서 읽기와 쓰기를 개선합니다. 주요 기능으로는 채팅, AI 사전, 번역기(이미지, PDF, 유튜브), OCR, 문법 확인, 웹페이지 내 쓰기 향상, 웹이미지 편집 등이 있습니다.

### Sider로 무엇을 할 수 있나요?

#### 웹페이지 요약

현재 보고 있는 웹페이지의 주요 내용을 한눈에 요약 제공합니다.

#### 질문 및 답변

웹페이지에 관련된 질문을 입력하면 AI가 실시간으로 답변합니다.

#### 작업 흐름 개선

다양한 AI 모델을 활용해 이메일 작성, 문서 초안 작성, 번역 등의 작업을 간소화합니다.

#### 자동 번역

방문 중인 페이지의 콘텐츠를 원하는 언어로 자동 번역합니다.

#### 브라우저 내 바로 사용

별도의 탭 전환 없이 브라우저 사이드바에서 직접 AI 모델을 사용합니다.

#### 맞춤형 제안

검색된 콘텐츠에 기반하여 관련 자료나 링크를 추천합니다.

#### 통합 워크플로 지원

정보를 검색하면서 요약과 작성까지 이어지는 통합된 작업을 지원합니다.

#### 파일 및 멀티미디어 지원

PDF, 이미지, 오디오 파일과의 상호작용이 가능합니다.

### Sider를 선택한 이유

#### 사용 편의성

브라우저에 확장 프로그램 설치만으로 쉽게 사용할 수 있습니다.

#### 생산성 향상

웹 탐색 중 필요한 작업을 즉각 수행하며 시간을 절약합니다.

#### 실시간 정보 접근

최신 AI 모델을 활용한 즉각적인 정보 제공과 문제 해결을 지원합니다.

#### 다국어 지원

글로벌 작업 환경에서 번역과 다국어 콘텐츠 생성을 지원합니다.

#### 유연한 통합

다양한 웹 애플리케이션과의 호환성으로 작업 흐름에 자연스럽게 통합됩니다.

#### 다양한 AI 모델 지원

GPT-4o, Claude 3.5, Gemini 1.5 등 다양한 AI 모델을 제공합니다.

#### 크로스 플랫폼 지원

Chrome, Edge, Safari 확장 프로그램뿐만 아니라 iOS, Android, Mac, Windows 앱으로도 사용할 수 있습니다.
